The reason for creating was not to be a protfolio at the first place. It started as a CSS practice/self teaching tutorial. Back in the Macromedia (later Adobe) Flash days, the websites were, just say - a bit different from today's websites. Everybody wanted to have a cool "Loading" screen, crazy shapes, graphics, animations. I wanted to see if those designs can be made suitable with latest technologies. And that's how it started.

First of all, the layout is not mine, it belongs to gregbike This is from 2007 and the design became a Daily Deviation - meaning one of the featured artwork from 10s and 1000s artworks posted at DA.

Secondly, I avoided using any image in this practice website. So, what you are looking at are all html elements.

Thirdly, this website is responsive. Well, isn't every other website now a days? Great CSS frameworks like Bootstrap, w3css makes life a lot easier. The initial release will contain w3css as css framework, but ultimately no framework/libraries will be used here except for Reactjs. So, no jQuery either. The reason for mentioning is, the mobile views are designed by me, keeping the original theme in mind.

Fourthly, since it started as a practice reservoir, I'd like to keep it that way. I am currently learning Reactjs, so I will be using some of the lessons here. That doesn't mean the contents of this website is true. The information I will be putting in here are all genuin.
